# Gestionnaire de joueurs

Une page de gestion des joueurs et de leurs scores, conçue pour être utilisée avec les autres jeux du projet (Morpion, Tux...). Elle centralise les scores et permet de suivre les performances des joueurs au fil des parties.

---

## Structure du projet

```
.
├── index.html    # Structure de la page
├── joueurs.css   # Mise en page du tableau
└── joueurs.js    # Gestion dynamique des joueurs
```

---

## Fonctionnalités

- Affichage de la liste des joueurs avec leurs scores dans un tableau
- Ajout d'un nouveau joueur via un formulaire (nom + score)
- Calcul automatique du **score total** mis à jour à chaque ajout

---

## Lancer la page

Ouvrir `index.html` dans un navigateur — aucune installation requise.
