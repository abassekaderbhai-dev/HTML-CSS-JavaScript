# Morpion 

Un jeu de morpion classique jouable à deux, développé en HTML, CSS et JavaScript.

---

## Structure du projet

```
.
├── index.html     # Structure de la grille
├── morpion.css    # Mise en page
└── morpion.js     # Logique du jeu
```

---

## Règles du jeu

- Le jeu se joue à **deux joueurs** sur le même appareil
- Les joueurs jouent chacun leur tour avec **X** et **O**
- Le premier à aligner **3 symboles** (en ligne, colonne ou diagonale) gagne
- Si toutes les cases sont remplies sans vainqueur, c'est une **égalité**
- Une case déjà jouée ne peut pas être rejouée

---

## Améliorations prévues

- Détecter automatiquement la victoire ou l'égalité
- Afficher le joueur dont c'est le tour
- Ajouter un bouton rejouer pour recommencer sans recharger la page
- Suivre le score sur plusieurs manches

---

## Lancer le jeu

Ouvrir `index.html` dans un navigateur — aucune installation requise.
